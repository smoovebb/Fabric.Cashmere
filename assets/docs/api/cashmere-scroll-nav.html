Module Import

import {ScrollNavModule} from '@healthcatalyst/cashmere';

Directives

HcScrollNavComponent

Container for scroll navigation links.

Selector: hc-scroll-nav

HcScrollNavContentComponent

Contains scrollable content that is navigable via hc-scroll-nav links.

Selector: hc-scroll-nav-content

Properties
Name Description
@Input()

bufferSpace: this.DEFAULT_BUFFER

Number in pixels, used to give a little leeway in the shifting of the active nav when scrolling. Defaults to 40. Example: Left at default, if showing just the bottom 40 pixels of the section before, count the next section as active.
@Input()

makeLastTargetFullHeight: true

If true, will force the height of the final scroll target area to be the height of the scrollable container. This is helpful if you want the last target in the content area to be able to scroll to the top. You can alternatively target the last item with css. Defaults to true.
@Input()

nav: HcScrollNavComponent

Reference to the scroll nav component.
@Input()

shouldAnimateScroll: true

If true, applies smooth scrolling via css. Defaults to true.
@Output()

newSectionInView: EventEmitter<string>

Fires when a new section is scrolled into view. Broadcasts the id of that section.

sectionInView: string

Id of the current section scrolled into view.
Methods
checkActiveSection()

Will update the navigation state.
refresh()

Scroll to top and reset the "automatic full height for the last item" setting.
scrollToTop()

Helper function to scroll to the top of the content area.

ScrollNavTargetDirective

Marks the host element as a linkable section within an hc-scroll-nav-content. Must set the id and link to it via an hcScrollLink.

Selector: [hcScrollTarget]

ScrollNavLinkDirective

Marks the host element as a link within an hc-scroll-nav.

Selector: [hcScrollLink]

Properties
Name Description
@Input()

hcScrollLink: string

The id of the corresponding hcScrollTarget that you would like to link to.
